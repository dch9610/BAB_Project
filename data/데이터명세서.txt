# 데이터 명세서

1. 중앙혈액원은 2010년 7월 1일자로 서울서부혈액원을 통합하고, 그 명칭을 서울서부혈액원으로 변경함
2. 2010년도 중앙혈액원 헌혈실적은 통합 전 중앙혈액원과 통합 이후 서울서부혈액원의 헌혈실적을 합한 값임
3. 2010년도 서울서부혈액원 헌혈실적은 통합 전 서울서부혈액원의 헌혈실적 값임


- 헌혈방법별 헌혈통계
 	1. 헌혈방법별 > 혈소판    => 백혈구여과제거
 	2. 헌혈방법별 > 다종성분 => 백혈구여과제거
 	
- 헌혈자 수
1. 자료(총 인구, 헌혈가능인구) : 통계청 추계인구
2. 인구수는 당시 추계인구를 바탕으로 작성한 것이며, 현재 추계인구와 다를 수 있음
3. 2009.1.30.일자로 헌혈이 가능한 나이가 만16세~69세로 종전 만64세보다 5년이 연장되어 2010년 통계부터는 이 기준을 반영함
4. 2019년 자료부터 '총인구' , '헌혈가능인구' 자료는 행정안전부 「주민등록인구현황」을 기준으로 작성함

- 헌혈공급실적
 	1. 항목 > 혈액량 => unit : 혈액제제(혈액을 원료로 하여 제조한 의약품)을 세는 단위
	2. 혈액공급실적(현황별) > 수혈용
		적혈구 : 농축적혈구 + 세척적혈구 + 백혈구여과제거적혈구
		혈소판 : 농축혈소판 + 혈소판풍부혈장 + 백혈구여과제거혈소판 + 세척혈소판
		혈장 : 신선동결혈장 + 동결혈장 + 동결침전제제 + 동결침전물제거혈장
		기타 : 전혈 + 농축백혈구
		
- 혈액 검사 부적격
	1. 부적격혈액검사별 > 부적격혈액검사 => 검사 부적격 중복으로 각 항목별 합계와 다를 수 있음
	2. 부적격혈액검사별 > 간기능검사       => 성분채혈혈장에는 실시하지 않음
	3.부적격혈액검사별 > 비예기항체검사  => 성분채혈혈장에는 실시하지 않음
	4. 부적격혈액검사별 > HTLV 항체검사 => 성분채혈혈장에는 실시하지 않음
